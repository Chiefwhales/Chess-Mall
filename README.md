Chief Whales Chess Mall
Chief Whales Chess Mall is a premium, four-page web application designed for chess enthusiasts and professionals. It features a modern, high-contrast aesthetic built with HTML and Tailwind CSS, focusing on elegance, strategy, and a seamless user experience.

Features
Premium Visuals: High-quality imagery with glass effects.

Fully Responsive: Mobile-first design with a custom-built hamburger menu for smaller screens.

Modern Tech Stack: Utilizes the latest Tailwind CSS 4.0 (Alpha/Browser builds) for rapid, utility-first styling.

Interactive Elements: Hover animations, scale effects on buttons, and subtle pinging notifications for "New Collections."

Four-Page Architecture:

Home: A high-impact landing page.

Chess Shop: Browsing and purchasing premium equipment.

About Us: Exploring the vision and story behind the brand.

Contact Us: Direct communication channels and social media integration.

Design Philosophy
The project uses a bg-slate-950 palette to evoke the deep focus required in a chess match. The use of "Indigo" and "Blue" accents represents the "Deep Sea/Whale" theme, bridging the gap between the majestic nature of the whale and the intellectual depth of chess.
